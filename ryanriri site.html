🔒 FDR – Sécuriser un serveur HTTP (réponses et déploiement)

📌 Réponses clés :
- Cryptographie symétrique vs asymétrique pour TLS.
- Certificat SSL contient clé publique, identité, dates validité.
- Handshake TLS : négociation chiffrement.
- Chain of Trust : autorité racine → intermédiaire → certificat serveur.
- Fichier de configuration : default-ssl.conf / sites-available
